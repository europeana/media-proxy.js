# Media proxy.js

Express app to proxy media from providers.

## Dependencies

* Node.js 16
* [Europeana API key](https://pro.europeana.eu/get-api)

## Install

```
npm ci
```

## Run for production (without .env support)

```
npm run start
```

## Run for development (with .env support)

```
npm run start:dev
```
